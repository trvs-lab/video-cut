# 卡顿词

## 模式

同一个**有意义的词**连续说 2-3 次：

```javascript
const stutterPatterns = [
  '那个那个',
  '就是就是',
  '然后然后',
  '这个这个',
  '所以所以'
];
```

**⚠️ 语气词特殊处理**：

| 案例 | 处理 | 说明 |
|------|------|------|
| 嗯嗯 | **不删** | 可能是强调语气 |
| 啊啊 | **不删** | 可能是感叹语气 |
| 呃呃 | 可删 | 纯卡顿 |
| 哦哦 | **不删** | 可能是领悟语气 |

**语气词一般不删**，因为：
1. "嗯嗯" = 强调肯定（如"嗯嗯，就是这样"）
2. "啊啊" = 感叹语气（如"啊啊，原来如此"）
3. 真正的卡顿后面会接其他词，如"嗯嗯然后..."

## 删除策略

删前面，保留最后一个。

```
原文: "那个那个我想说"
删除: "那个"
保留: "那个我想说"
```
