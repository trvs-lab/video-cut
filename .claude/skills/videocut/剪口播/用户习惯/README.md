# 用户习惯

用户个人偏好，AI 审核时参考。

## 文件清单

| 文件 | 类型 | 内容 |
|------|------|------|
| 1-核心原则.md | 原则 | 删前保后 |
| 2-语气词检测.md | 偏好 | 嗯啊呃 + 删除边界 |
| 3-静音段处理.md | 阈值 | ≤0.5s忽略, 0.5-1s可选, >1s建议删 |
| 4-重复句检测.md | 偏好 | 相邻句子开头≥5字相同，删短的 |

| 9-残句检测.md | 偏好 | 话说一半被打断 |

## AI 审核顺序（按优先级）

1. **静音 >1s** → 建议删（按1秒格子拆分）
2. **残句** → 删（话说一半 + 静音）
3. **重复句** → 删短的（开头≥5字相同）
4. **句内重复** → 删 A+中间（A+中间+A 模式）
5. **卡顿词** → 删前面（那个那个、就是就是）
6. **重说纠正** → 删前面（部分重复、否定纠正、词被打断）
7. **语气词** → 标记待人工确认（嗯啊呃）

## 核心原则

**删前保后**：后说的通常更完整，删除前面保留后面。
